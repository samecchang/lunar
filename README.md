# Traditional Chinese & Lunar Calendar

A modern, responsive web-based calendar that seamlessly integrates the Gregorian calendar with the traditional Lunar calendar, optimized for Traditional Chinese users.

## Features
*   **Dual Calendar View**: Side-by-side display of Gregorian dates and their corresponding Lunar dates.
*   **Traditional Chinese UI**: Fully localized interface using Traditional Chinese characters for cultural accuracy.
*   **100% Client-Side**: No database or server required; runs entirely in your browser for fast performance.
*   **Responsive Design**: A clean grid layout that works perfectly on both desktop and mobile devices.

## Built With
*   **HTML5 / CSS3**: For a minimalist and functional user interface.
*   **Vanilla JavaScript**: Core logic for date calculations and Lunar-to-Solar conversion.

## How to Use
1.  **Clone** or download this repository.
2.  Open `LunarCalendar.html` in any modern web browser.
3.  **Navigate** through months and years using the header controls.
4.  View specific Lunar details and Solar Terms directly on the date tiles.
